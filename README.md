A full-stack job portal application built using the MERN stack with
authentication, admin dashboard, and job application features.

## Features
- User & Recruiter authentication (JWT)
- Admin dashboard
- Job posting & job applications
- Redux Toolkit for state management
- File uploads using Multer
- Protected routes

## Tech Stack
- MongoDB
- Express.js
- React (Vite)
- Node.js
- Redux Toolkit

## Run Locally

### Frontend
cd frontend/vite-project  
npm install  
npm run dev  

### Backend
cd backend  
npm install  
npm run dev
